# Create New Pristine Seas Project Structure

Creates a simple standardized folder structure for Pristine Seas
projects in the projects folder of the science drive.

## Usage

``` r
create_project(project_name)
```

## Arguments

- project_name:

  Character string. Name of the new project (use "prj-" prefix by
  convention)

## Value

Character string. Path to the created project folder

## Examples

``` r
if (FALSE) { # \dontrun{
# Create a project
create_project("prj-coral-reef-assessment")

# Create another project
create_project("prj-fish-surveys")
} # }
```
